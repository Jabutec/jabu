# Personal Portfolio Website

This is my personal portfolio website where I showcase my projects, technical skills, and background as a recent BSc Information Technology graduate. The goal of this website is to highlight my work, share the projects I have built, and provide a central place where recruiters and collaborators can learn more about me.

## Features

- About Me section introducing my background and interests
- Projects page showcasing my work
- Downloadable CV
- Contact page for professional inquiries

## Technologies Used

- HTML
- CSS
- JavaScript

## Purpose

This website serves as my professional online presence where I document my learning journey and demonstrate my interest in technology, particularly in data analytics and software development.
